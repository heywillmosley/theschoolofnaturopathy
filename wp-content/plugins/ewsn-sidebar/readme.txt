=== EWSN Sidebar ===
Contributors: @heywillmosley
Requires at least: 5.0
Tested up to: 5.3
Requires PHP: 7.0

Sidebar that displays recommended book and product suggestions based on the user\'s progress in the School\'s tracks, courses and lessons.

== Installation ==
After activation:
1. Go to Widgets.
2. Add Text element.
3. Add shortcode \'[ewsn-sidbar] and click save.

You\'ll see the sidebar display on Track, Course and Lesson pages with relevant content.
